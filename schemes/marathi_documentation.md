# योजना शोधक प्रोजेक्टचे मराठी दस्तऐवज

हा प्रोजेक्ट महाराष्ट्रातील नागरिकांसाठी सरकारी योजनांची माहिती, पात्रता तपासणी आणि अर्ज प्रक्रियेची सोपी माहिती देण्यासाठी तयार करण्यात आला आहे. या प्रोजेक्टमध्ये आम्ही React, Vite, Node.js, Express, PostgreSQL आणि JSON डेटा वापरून एक पूर्ण व्यावहारिक संकेतस्थळ तयार केले आहे.

---

## 1. या प्रोजेक्टमध्ये आपण काय केले?

आम्ही या प्रोजेक्टमध्ये खालील गोष्टी लागू केल्या आहेत:

### 1.1 सरकारी योजनांची यादी आणि माहिती
- विविध सरकारी योजनांची माहिती एकाच ठिकाणी दर्शविण्यासाठी पृष्ठ तयार केले.
- प्रत्येक योजनेची थोडक्यात ओळख, लाभ, पात्रता निकष आणि कागदपत्रे दाखवली जातात.
- योजनेच्या तपशीलांसाठी वेगवेगळे dynamique पृष्ठ तयार केले.

### 1.2 पात्रता तपासणी प्रणाली
- महिलांसाठी, शेतकऱ्यांसाठी आणि ज्येष्ठ नागरिकांसाठी स्वतंत्र पात्रता तपासणी पृष्ठे तयार केली.
- अर्जदाराच्या माहितीच्या आधारे त्याला कोणत्या योजना योग्य आहेत हे तपासण्याची सुविधा जोडली.
- थेट निकषांसोबतच अप्रत्यक्ष निष्कर्षही दाखवले आहेत.

### 1.3 लॉजिक ट्री आधारित तपासणी
- प्रत्येक योजनेसाठी पात्रतेचा नियम आणि विचारप्रक्रिया तयार केली.
- वापरकर्ता जो प्रतिसाद देतो त्यावर आधारित प्रणाली त्याला पात्र/अपात्र ठरवते.
- यामध्ये वय, लिंग, उत्पन्न, सरकारी नोकरी, आयकरदाता, वाहन, पक्के घर, सौर पंप इत्यादी निकषांचा समावेश आहे.

### 1.4 युझर इंटरफेस
- एक आकर्षक होमपेज, नेव्हिगेशन बार, फूटर आणि योजना कार्ड्स तयार केले.
- योजना शोधण्यासाठी सर्च बार दिला.
- अर्ज करण्यासाठी, माहिती पाहण्यासाठी आणि संपर्कासाठी स्वतंत्र दुवे दिले.

### 1.5 माझे अर्ज आणि संपर्क पृष्ठे
- “My Applications” पृष्ठावर अर्ज केलेल्या योजनांची माहिती दाखवण्याची रचना केली.
- “Contact” पृष्ठावर संपर्क फॉर्म, माहिती आणि सहाय्यात्मक दुवे तयार केले.

---

## 2. प्रोजेक्टची मुख्य रचना

### 2.1 फ्रंटएंड
- React + Vite वापरून UI तयार केले.
- React Router द्वारे होम, योजनेचे तपशील, पात्रता तपासणी, माझे अर्ज आणि संपर्क या पृष्ठांमध्ये नेविगेशन केले.
- Lucide React आयकन्स वापरून इंटरफेस अधिक आकर्षक आणि वापरण्यास सुलभ केला.

### 2.2 बॅकएंड
- Node.js आणि Express वापरून API तयार केले.
- सरकारी योजनांची माहिती, योजनेचे तपशील आणि पात्रता तपासणीची माहिती API द्वारे उपलब्ध करून दिली.
- API द्वारे frontendला डेटा मिळतो.

### 2.3 डेटाबेस
- PostgreSQL डेटाबेस वापरून योजनांची माहिती संरक्षित केली.
- यामध्ये योजना, पात्रता निकष, फायदे, कागदपत्रे, अपात्रतेचे कारणे आणि अर्ज प्रक्रियेतील पायऱ्या यांचा समावेश आहे.

---

## 3. आमच्या प्रोजेक्टमध्ये समाविष्ट केलेले प्रमुख घटक

### 3.1 होम पेज
- युझरला सर्वात महत्त्वाच्या सरकारी योजनांची माहिती मिळते.
- शोध बार, सूचना पट्टी, लोकप्रिय श्रेणी आणि योजना कार्ड्स आहेत.
- योजनेवर क्लिक केल्यास तपशील पृष्ठावर जाऊन माहिती पाहता येते.

### 3.2 योजनेचे तपशील पृष्ठ
- प्रत्येक योजनेच्या पात्रता निकषांची माहिती दिली जाते.
- फायदे, कागदपत्रे आणि अर्ज करण्याची प्रक्रिया दर्शविली जाते.
- पात्रता लॉजिक ट्री टॅबमध्ये अर्जदाराच्या उत्तरांवर आधारित निकाल दाखविण्यात येतो.

### 3.3 पात्रता तपासणी पृष्ठे
- महिलांच्या योजनांसाठी, शेतकरी योजनांसाठी आणि ज्येष्ठ नागरिकांच्या योजनांसाठी स्वतंत्र फॉर्म्स आहेत.
- या फॉर्ममध्ये प्रश्नांची उत्तरे देऊन युझरला त्याच्या पात्रतेची माहिती मिळते.

### 3.4 नियम इंजिन
- ही प्रणाली थेट नियम आणि अप्रत्यक्ष नियम यांच्याद्वारे कार्य करते.
- उदाहरणार्थ, सौर पंप किंवा RCC घर असल्यास, तो विशिष्ट योजनांसाठी अपात्र ठरू शकतो.
- यामुळे “Indirect Eligibility Impact” किंवा “अप्रत्यक्ष प्रभाव” दाखवता येतो.

---

## 4. आम्ही कोणत्या डेटाशी काम केले?

या प्रोजेक्टमध्ये सरकारी योजनांची माहिती JSON फाईल्समध्ये उपलब्ध होती. ती माहिती पुढील प्रकारे वापरली गेली:
- योजना नाव
- संक्षिप्त माहिती
- उद्देश
- पात्रता निकष
- लाभ
- कागदपत्रे
- अर्ज प्रक्रिया

या सर्व डेटा कडे आपण backendमध्ये import करून database मध्ये साठवले.

---

## 5. बॅकएंडने काय केले?

### 5.1 API endpoints
- सर्व योजनांची यादी मिळविण्यासाठी API तयार केला.
- विशिष्ट योजनेचा तपशील मिळविण्यासाठी API तयार केला.
- योजनेच्या अपात्रतेचे कारणे वेगवेगळ्या Endpoint द्वारे देण्यात आले.
- अर्जदाराच्या माहितीवर आधारित पात्रता तपासणीसाठी API जोडला.

### 5.2 Rules Engine
- rulesEngine.js मध्ये वेगवेगळ्या योजनांसाठी नियम दिले आहेत.
- या नियमांमध्ये खालील बाबी समाविष्ट आहेत:
  - वय
  - लिंग
  - उत्पन्न
  - आयकरदाता 여부
  - सरकारी कर्मचारी 여부
  - चारचाकी वाहन
  - पक्के घर
  - सौर पंप/सोलर योजना

## 6. प्रोजेक्टचे मुख्य फायदे

- सरकारी योजनांची माहिती एकाच ठिकाणी उपलब्ध होते.
- वापरकर्ता सहजपणे पात्रता तपासू शकतो.
- प्रकल्प मराठी आणि इंग्रजी दोन्ही भाषेत सुलभपणे वापरला जाऊ शकतो.
- डेटा डाटाबेसमध्ये संरक्षित असल्यामुळे ते नंतर वाढवता येते.
- डाटाबेस बंद असताना किंवा API उपलब्ध नसताना देखील fallback data वापरून वेबपेज कार्यरत राहतो.

## 7. हे प्रोजेक्ट कसे चालवायचे?

### 7.1 बॅकएंड सुरू करणे
- backend folder मध्ये जा.
- npm install करा.
- npm start चालवा.

### 7.2 फ्रंटएंड सुरू करणे
- yojana-shodhak folder मध्ये जा.
- npm install करा.
- npm run dev चालवा.

### 7.3 ब्राउझरमध्ये पाहणे
- सामान्यतः http://localhost:5173 या पत्त्यावर प्रोजेक्ट उघडता येतो.

## 8. प्रोजेक्टचे सारांश

या प्रोजेक्टमध्ये आपण सरकारी योजनांची माहिती, पात्रता तपासणी, लॉजिक ट्री आधारित विश्लेषण, डाटाबेस आणि API असलेले एक पूर्ण डिजिटल पोर्टल तयार केले आहे. याचा मुख्य उद्देश नागरिकांना त्यांच्या घरबसल्या सरकारी योजनांची माहिती मिळवून योग्य योजना ओळखण्यास मदत करणे आहे.

सारांशतः, या प्रोजेक्टमध्ये आपण:
- सरकारी योजना पोर्टल तयार केले,
- पात्रता तपासणी प्रणाली जोडली,
- नियम आधारित लॉजिक ट्री तयार केली,
- डेटा डाटाबेसमध्ये साठवला,
- React frontend आणि Node.js backend जोडले,
- युझर-फ्रेंडली अनुभव देणारे पृष्ठे विकसित केली.

- SMS किंवा ईमेल सूचना सुविधा जोडणे.

- वापरकर्ता लॉगिन आणि अर्ज इतिहास सिस्टीम जोडणे.
